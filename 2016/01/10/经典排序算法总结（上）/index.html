经典排序算法总结（上）

排序算法总结
大二曾学过数据结构和常用算法，现在回忆起来还稍有印象。
经典的排序算法有插入排序、计数排序、选择排序、堆排序、冒泡排序、快速排序、归并排序、基数排序，现在来一一回忆。

插入排序
插入排序的思想就是，给定一组乱序的数，先将第一个数放进一个数据组中。再选出第二个数，与第一个数相比，顺序排好。再选出第三个数，与依次与前两个数相比，小于就放在之前。以此类推。

最优复杂度：O(n) 顺序
最差复杂度：O(n2) 倒序

代码如下：

for(int i=1;i<a.length;i++){
    int j=i-1;
    //将取出数储存
    int key=a[i];
    while(j>=0&&a[j]>key){
        a[j+1]=a[j];
        a[j]=key;
        j--;
    }            
}

冒泡排序
最优复杂度：O(n)（改进）
复杂度：O(n2)（普通）
个人感觉和插入排序挺相似的，冒泡顾名思义，小泡比大泡先冒出水面。只需要选一个方向，然后顺序的选一个数，不断的往后选择相邻的数进行比较，位置不对就交换，直到最后一个数。

代码如下：

//保证每个数都选择比较过
for(int i=0;i<a.length;i++){
    //每次被比较的n-1个数
    for(int j=0;j<a.length-1;j++){
        if(a[j]>a[j+1]){
            int c=a[j+1];
            a[j+1]=a[j];
            a[j]=c;

        }
    }
}

改进的方法可以设一个标签，用于记录每次排序停止的位置，因为第一次就可以确定最大值，每一次至少可以确定一个剩下的最大值。
代码如下：

int label=a.length-1;
while(label>0){
    //每次被比较的n-1个数
    int pos=0;
    for(int j=0;j<label;j++){
        if(a[j]>a[j+1]){
            pos=j;                    
            int c=a[j+1];
            a[j+1]=a[j];
            a[j]=c;
            d++;                
        }                            
    }
    label=pos;            
}

快速排序
最优复杂度：O(nlogn)（改进）
最差复杂度：O(n2)（普通）


代码如下：

public static int [] a={10,9,4,4,1,45,23,4,6};
public static void main(String[] args) {
    // TODO Auto-generated method stub

    int high=a.length-1;  //8
    int low=0;
    quick(a,low,high);

    for(int i=0;i<a.length;i++){
        System.out.println(a[i]);
    }
}

private static void quick(int[] a, int low, int high) {
    // TODO Auto-generated method stub
    if(low<high){
        int  c=quicksort(a,low,high);
            //递归用的好啊
        quick(a, low, c-1);
        quick(a, c+1, high);
    }
}

private static int quicksort(int[] a,int low,int high) {
    // TODO Auto-generated method stub
    int first=a[low];  
    while(low<high){        
        while(low<high&&a[high]>=first) {
            high--;            
        }        
        int swap1=a[high];
        a[high]=first;
        a[low]=swap1;    
        while(low<high&&a[low]<=first){
            low++;
        }        
        int swap2=a[low];
        a[low]=first;
        a[high]=swap2;            
    }

    QuickSort.a=a;
    return low;
}

归并排序
复杂度：O(nlgn)
将两组数据按大小排列成一组数据，如果事先给出了一组乱序数据需要排列，就可以把每一个单个的数据看成是一组，组合成两倍于原长的数据，再选择2组2倍长度的数据进行组合，直到最终排列完成。

代码实现起来比较费劲，尝试了好久，还是参考了下牛人的才能写出：

public static void sort(int[] data, int left, int right) {
        // TODO Auto-generated method stub
        if(left<right){
            //找出中间索引
            int center=(left+right)/2;
            //对左边数组进行递归
            sort(data,left,center);
            //对右边数组进行递归
            sort(data,center+1,right);
            //合并
            merge(data,left,center,right);

        }
    }
    public static void merge(int[] data, int left, int center, int right) {
        // TODO Auto-generated method stub
        int [] tmpArr=new int[data.length];
        int mid=center+1;
        //third记录中间数组的索引
        int third=left;
        int tmp=left;
        while(left<=center&&mid<=right){
            //从两个数组中取出最小的放入中间数组
            if(data[left]<=data[mid]){
                tmpArr[third++]=data[left++];
            }else{
                tmpArr[third++]=data[mid++];
            }
        }
        //剩余部分依次放入中间数组
        while(mid<=right){
            tmpArr[third++]=data[mid++];
        }
        while(left<=center){
            tmpArr[third++]=data[left++];
        }
        //将中间数组中的内容复制回原数组
        while(tmp<=right){
            data[tmp]=tmpArr[tmp++];
        }
        System.out.println(Arrays.toString(data));
    }
