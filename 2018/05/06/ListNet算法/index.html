ListNet算法

ListWise方法

ListWise方法是将一次请求的list of objects作为一条样本进行训练，所有请求可以表示为：$Q= \left ( q^{\left (1\right )}, q^{\left (2\right )}, … , q^{\left (m\right )} \right )$，每一个请求中的文档可以用$d^{\left (i\right )}= \left ( d_{1}^{\left (i\right )}, d_{2}^{\left (i\right )}, … , d_{n^{i}}^{\left (i\right )} \right )$其中$d_{j}^{(i)}$表示第j个文档、n表示$d^{(i)}$的数量，对应每个文档会有一个相应的得分$y^{\left (i\right )}= \left ( y_{1}^{\left (i\right )}, y_{2}^{\left (i\right )}, … , y_{n^{i}}^{\left (i\right )} \right )$，$x_{j}^{(i)} = \Psi (q^{(i)},d_{j}^{(i)})$表示每个query-document对$(q^{(i)},d_{j}^{(i)}),i = 1,2,…,m;j=1,2,…,n^{(i)}$的特征向量，每次请求的特征列表$x^{\left (i\right )}= \left ( x_{1}^{\left (i\right )}, x_{2}^{\left (i\right )}, … , x_{n^{i}}^{\left (i\right )} \right )$及对应的得分$y^{\left (i\right )}= \left ( y_{1}^{\left (i\right )}, y_{2}^{\left (i\right )}, … , y_{n^{i}}^{\left (i\right )} \right )$，对任意的$x^{(i)}$使得$z^{(i)} = (f(x_{1}^{(i)}),…,f(x_{n^{(i)}}^{(i)}))$，形式化最小化训练数据的损失得出损失函数：

排列分布

$\pi = <\pi(1),\pi(2),…,\pi(n)>$表示一次排列，$\pi(j)$表示排列在位置 $j$，所有可能的数量为n的排列的组合用$\Omega_{n}$表示。经过ranking模型的排序后，我们使用$s = (s_{1},s_{2},…,s_{n})$表示得分。

定义

$\pi$表示某一个排列，$\phi(.)$是一个值域为正数的单调递增函数。对于某一个排列$\pi$以及相应的得分s，定义排列的概率为：

其中，$s_{\pi(j)}$表示排列$\pi$位于位置j的对象的得分。
以上公式的性质：

  • 所有排列的概率之和为1
  • 概率最大排列是按照得分逆序，最小的是升序
  • 交换排列中两个的位置，得分高的前移会使得概率增加
  • 如果$\phi(x)=ax$是一个线性函数，可以保证$p_{s}(\pi)$缩放不变性
  • 如果$\phi(x)=exp(x)$是一个指数函数，可以保证$p_{s}(\pi)$平移不变性
