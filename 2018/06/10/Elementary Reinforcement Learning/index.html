Elementary Reinforcement Learning

强化学习的要素

  • 奖励（Reward）
  • 个体（Agent）
  • 环境（Environment）
  • 马儿可夫属性（Markov Property）
  • 序列决策

强化学习是由以上要素组成，表现形式为Agent在给定环境和时间$t$情况下，不断的采取行为$A_{t}$改变环境状态并获得奖励，来不断增强下一步行为的过程。

在t 时刻有一个对环境的评估$O_{t}$，个体做出一个行为$A_{t}$，并从当前环境$S_{t}$，中获取一个奖励$R_{t+1}$，该奖励可以理解为离开$S_{t}$获得的即时奖励。此时，环境接收到个体的动作$A_{t}$，更新环境进入$S_{t+1}$，同时个体得到下一个观测$O_{t+1}$。
强化学习的样本就是由一系列观测、个体、状态组合而成的:

强化学习个体的组成

  • 策略（Policy）
    • 策略代表着个体的行为
    • 是状态到行为的映射
    • 表示为：$a = \pi (s)$
    • 随机策略表示为：$\pi (a|s) = \mathbb{P}[A_{t}=a|S_{t}=s]$
  • 价值函数（Value Function）
    • 值函数是对未来奖励的预测
    • 用来预测当前状态的好与坏的程度
    • 因此某一次选择的action的价值为当前奖励和后续所有奖励的累加:
    • $v(\pi) = \mathbb{E}_{\pi}[R_{t+1} + \gamma R_{t+2} + \gamma ^{2}R_{t+2} + …]$
    • 由于对未来的不确定性，使用discount future reward来表示$R_{t} = r_{t} + \gamma R_{t+1}$
  • 模型（Model）
    • 模型用于预测环境
    • 给定当前状态和动作，预测下一个状态
    • $P^{a}_{ss^{‘}} = \mathbb{P}[S_{t+1}=s^{‘}|S_{t}=s, A_{t} = a]$
    • $R_{s}^{a} = \mathbb{E}[R_{t+1}|S_{t}=s, A_{t} = a]$

强化学习分类

  • Value Based
    • No Policy
    • Value Function
    • 策略函数由价值函数间接得到
  • Policy Based
    • No Value Function
    • 策略直接产生行为
  • Actor Critic
    • Policy
    • Value Function

或者：

  • Model Free
    • Policy or / and Value Function
    • 个体不知道环境运行机制
  • Model Based
    • Policy or / and Value Function
    • 个体知道环境运行机制

学习和规划

  • Reinforcement Learning

    • 初始环境未知
    • 环境和个体相互作用
    • 逐渐改善个体行为

  • Planing

    • 环境的Model已知
    • 个体不与环境交互，而是通过和模型的交互来改善策略

探索和利用

  • Exploration

    • 不断试错发现更好的策略
    • 且不在试错过程中丢失太多的奖励

  • Exploitation

    • 探索过程中发现了许多环境的信息
    • 利用信息去寻找最大的奖励
    • 平衡探索和利用非常重要

预测和控制

  • Prediction

    • 通过给予的策略估计未来
    • 给定策略下求解价值函数的过程

  • Control

    • 寻找最优策略来最大化未来的奖励
